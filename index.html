Домашнє завдання: GIT

  • git config --global user.name blabla - встановлення імені користувача
  • git config --global user.email blabal@bla - встановлення пошти користувача
  • git init - ініціювати стеження за змінами
  • git status - поточний статус
  • git add - додати файли
  • git commit - створити коміт
  • git log - перегляд змін в проекті
  • git checkout - зміна поточної гілки
  • git branch - створити гілку
  • git push - запушити на віддалений репозиторій
  • git pull - стягнути останні зміни на локальне місце роботи
  • git merge - вносить зміни з вказаної гілки в поточну
  • git clone - скачує репозиторій з всіма змінами
  • .gitignore файл - список файлів та папок, які не відслідковуються
